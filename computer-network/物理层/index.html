物理层

概念

  1. 常用一个固定时长的信号波形(数字脉冲)表示一位k进制数字,代表不同离散数值的基本波形就被叫做码元
  2. 码元传输速率(调制速率或符号速率) - 单位时间内传输的码元数，单位是波特。
  3. 数字编码的五种形式
    1. 非归零码 - 整个码元在x信号周期内保持同一电平
    2. 归零码 - 码元在传输过程中，信号在码元中间回到零电平
    3. 反向归零码 - 电平的跳变表示0，不变表示1
    4. 曼彻斯特编码 - 每个信号中心必跳变，码元的中心位置跳变表示0，不变表示1
    5. 差分曼彻斯特编码 - 仅作为时钟信号，不表示数据。在周期刚开始时根据0或1决定是否跳变
  4. 数据的调制形式 -
    1. 调幅 - 载波的振幅随基带信号而变化
    2. 调频 - 载波的频率随基带信号而变化
    3. 调相 - 载波的相位随基带信号而变化
    4. 正交幅度调制 - R=Blog2(mn), 其中R表示数据传输速率，B表示波特率，采用m个相位，每个相位有n种振幅
    5. 物理层接口四种特性 - 机械、电气、功能、过程
    6. 设备 -中继器(连接网段而不是子网)、集线器
      ## 公式
  5. 携带n个比特的码元 - 1Band = n b/s
  6. 奈奎斯特定理 - 理想低通信道下的数据下的极限传输速率 = 2Wlog2V，其中W表示信道的频率带宽，V表示每个码元的离散电平数目
  7. 香农定理 - 信道的极限数据传输速率 = Wlog2(1+S/N)，其中W表示信道的带宽，S/N表示信道内的信噪比，N表示信道内的噪声功率
