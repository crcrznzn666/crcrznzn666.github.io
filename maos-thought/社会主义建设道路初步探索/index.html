社会主义建设道路初步探索

发表于 2025-07-22 分类于 maos-thought

社会主义改造理论

定论

  1. 两次结合
    1. 第一次即马克思主义同中国新民主主义革命的融合
    2. 第二次即与中国的社会主义建设结合，探索社会主义建设道路
  2. 《论十大关系》的报告
    1. 明确提出要以苏为鉴、独立自主探索适合中国情况的社会主义建设道路
    2. 明确了建设社会注意必须根据本国情况走自己道路这一根本思想
    3. 确定基本方针 - 调动一切积极因素为社会主义服务
    4. 标志着党探索社会主义建设道路的良好开端
  3. 正确认识和处理社会主义矛盾的思想
    1. 1957年所做《关于正确处理人民内部矛盾的问题》
    2. 主要内容
      1. 矛盾普遍存在，这些矛盾推动社会主义不断向前发展
      2. 社会主义的基本矛盾，生产力与生产关系、上层建筑与经济基础的关系
        1. 是非对抗性的矛盾
        2. 相适应又相矛盾，相适应是基本方面
        3. 可以通过社会主义本身不断得到解决
      3. 两类不同性质的矛盾
        1. 敌我矛盾是根本利益对立基础上的矛盾,因此是对抗性矛盾
        2. 人民矛盾是人民利益根本一致条件下的矛盾,因此是非对抗性矛盾
        3. 解决敌我矛盾用专政的方法。解决人民内部矛盾用民主的方法
        4. 社会主义国家政治生活的主题即正确处理人民内部的矛盾
  4. 中共八大
    1. 正确分析了社会主义改造完成后我国社会主要矛盾的变化，明确提出新形势下党和人民的主要任务
    2. 我国主要矛盾已经不再是工人阶级和资产阶级的矛盾，而是人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾
    3. 主要任务是集中力量发展社会生产力，根本任务是在新的生产关系下保护和发展生产力
    4. 提出既反保守又反冒进，即在综合平稳中稳步前进的经济建设方针
    5. 肯定陈云提出的“三个主体、三个补充”的思想
    6. 政治建设上提出扩大社会主义民主，健全社会主义法治
  5. 走中国工业化道路的思想
    1. 《关于正确处理人民内部矛盾的问题》中，明确提出中国工业化道路的问题，主要是重工业和轻工业、农业的发展关系问题
    2. 内容 - 以农业为基础，以工业为主导，以农轻重为序发展国民经济的总方针，以及一系列“两条腿走路”的工业化发展思路(重工业和轻工业同时并举、中央工业和地方工业同时并举、沿海工业和内地工业同时并举、大型企业和小型企业同时并举等)
  6. 初步探索的其他理论成果
    1. 社会发展阶段 - 发达的社会主义和不发达的社会主义
    2. “四个现代化”的战略目标和战略步骤 -
    3. 关于教育文化和知识分子工作
    4. 发展商品生产、利用价值规律的思想，提出”两参一改三结合”
    5. 关于国防建设
    6. 关于祖国统一
    7. 关于国际战略和外交工作
    8. 关于执政党建设
  7. 初步探索的意义
