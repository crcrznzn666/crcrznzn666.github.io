中国特色社会主义理论体系的形成与发展

  1. 国际背景 - 和平与发展成为了时代主题
  2. 邓小平理论是中特理论的开篇之作
    1. 1982年党的十二大，邓小平理论的主题形成
    2. 1987年党的十三大，邓小平理论的轮廓形成
    3. 1992年邓小平南方谈话，是邓小平理论的集大成之作
    4. 1997年党的十五大，正式提出邓小平理论这一概念。确立为党的指导思想并写入党章
  3. 中国特色社会主义的历史地位

邓小平理论

  1. 基本的首要的理论问题 - 什么是社会主义、怎样建设社会主义(关键是坚持社会主义基本制度的基础山进一步认清社会主义的本质)
  2. 解放思想、实事求是是邓小平理论的精髓
  3. 社会主义初级阶段理论
    1. 必须经历很长的初级阶段趋势线其他国家在资本主义条件下实现的工业化和生产的商品化、社会化、现代化
    2. 我国已经进入社会主义社会，必须坚持不离开社会主义社会社会
    3. 我国社会主义社会还处在不发达的阶段、必须正式而不能超越这个阶段
  4. 党在社会主义初级阶段的基本路线概括为“一个中心，两个基本点”
    1. 根本任务问题 - 以经济建设为中心
    2. 政治保证问题 - 坚持四项基本原则
    3. 社会主义的发展动力和外部条件问题 - 坚持改革开放
    4. 奋斗目标 - 建设富强民主文明和谐的社会主义现代化国家
    5. 实现奋斗目标的领导和依靠力量 - 领导和团结各族人民
    6. 实现奋斗的根本立足点 - 自力更生、艰苦创业
  5. 社会主义根本任务和发展战略理论
    1. 社会主义的根本任务是发展生产力。党和国家的工作重点是经济建设
    2. 三步走基本实现现代化的发展战略
  6. 社会主义改革开放理论
    1. 十一届三中全会开启了改革开放和社会主义现代化的伟大征程
    2. 改革是社会发展的直接动力
    3. 判断改革和各方面是非得失的“三个有利于”标准 - 是否有利于发展社会主义生产力(首要最根本的一条)、是否有利于增强社会主义国家的综合国力、是否有利于提高人民生活水平
    4. 开放也是改革
    5. 对外开放要处理好的关系
    6. 改革开放史决定中国命运和实现中华民族伟大复兴的关键一招
  7. 社会主义市场经济理论
    1. 市场经济与计划经济不再对立，也不再是划分社会制度的标准
    2. 为十四大明确把建立社会主义市场经济体制作为我国经济体制改革的目标奠定了理论基础
    3. 计划和市场都是经济手段、作为资源分配的手段本身不具有制度属性、可以和不同的社会制度结合并表现出不同的性质
  8. 两手抓。两手都要硬
  9. 一国两制和祖国统一
  10. 中国特色社会主义外交和国际战略
  11. 邓小平；理论的历史地位

三个代表重要思想

  1. 中国共产党始终代表中国先进生产力的发展要求、代表中国先进文化的前进方向、代表最广大人民的根本利益
  2. 使我们党的立党之本、执政之基、力量之源
  3. 主要内容
    1. 发展史党执政兴国的第一要务
    2. 建立社会主义市场经济体制
    3. 全面建设小康社会 - 十五大报告提出三步走战略
    4. 建设社会主义政治文明
    5. 实施引进来和走出去结合的对外开放战略
    6. 推进党的建设新的伟大工程
  4. 三个代表重要思想的历史地位

科学发展观

  1. 第一要义是推动社会经济发展、核心立场是以人为本、基本要求是全面协调可持续，根本方法是统筹兼顾
    1. 加快转变经济发展方式
    2. 发展社会主义民主政治
    3. 推进社会主义文化强国建设
    4. 建设社会主义和谐社会(社会和谐是中特的本质属性)
    5. 推进生态文明建设
    6. 全面提高党的建设科学化水平
  2. 科学发展观的历史地位
