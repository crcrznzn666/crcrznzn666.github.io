中华民族的抗日战争

发表于 2025-08-18 更新于 2025-08-19 分类于 modern-history

1

历史

日本灭亡中国的计划及实施

  1. 事件
    1. 1931年918事变 - 日本侵略战争由此开始
    2. 1932年2月 - 中国东北全境沦陷
    3. 1935年华北事变 - 策动花呗无声两市防共自治活动，制造傀儡政权
    4. 1937年卢沟桥事变 - 日本全面侵华战争的开始
  2. 推行以华制华和以战养战的策略
  3. 日本帝国主义的残暴统治

中国共产党举起抗日武装的旗帜、

  1. 卢沟桥事变后中国在东方开辟了世界上第一个大规模的反法西斯战场
  2. 抗日救亡运动的兴起 - 1932年4月15日，中华苏维埃共和国临时中央政府宣布对日作战

抗日民族统一战线的建立与全民族抗战的开始

  1. 一二九运动 - “反对华北自治”，“打倒日本帝国主义”，“停止内战、一致对外”。促进了中华民族的觉醒，抗日救亡预定新高潮的到来
  2. 中共抗日民族统一战线新政策
    1. 1935年8月1日八一宣言 - 主张停止内战，组织国防政府和抗日联军，对日作战
    2. 1935年瓦窑堡会议 - 批评左倾错误和关门主义错误，提出党的基本策略任务是建立广泛的抗日民族统一战线
    3. 1936年《停战议和一致抗日通电》 - 放弃反蒋口号，明确总方针是“逼蒋抗日”
  3. 西安事变
    1. 1936年张学良杨虎城扣押蒋介石
    2. 西安事变的和平解决 - 标志着十年内战局面基本结束
    3. 1937年2月，中共中央致电国民党五届三中全会，提出停止内战、一致对外等五项要求
  4. 国共合作、共赴国难
    1. 1937年8月，红军改名为国民革命军第八路军，红军游击队改名为新四军
    2. 1937年9月，国民党中央通讯社发表《中共中央为公布国共合作宣言》，以国共两党第二次合作为基础的抗日民族统一战线正式形成

战略防御阶段和战略相持阶段的正面战场

  1. 1937年卢沟桥事变到1938年广州、武汉失守，中国抗战处于战略防御阶段
  2. 战略防御阶段正面战场的作用 - 台儿庄战役
  3. 国民党正面战场失利的原因
    1. 客观原因是日军在力量占比上有很大优势
    2. 主观原因是国民党战略指导方针上的失误
      1. 实行片面抗战的路线，不敢放手和发动民众，将希望单方面寄托在政府和正规军的抵抗上
      2. 战略战术上没有采取积极防御的方针，而是进行单纯的阵地防御战
  4. 战略相持的正面战场
    1. 有片面抗战转变为消极抗战 - 1939年国民党五届五中全会，防共限共的方针
    2. 太平洋战争爆发后配合英美打击日军
    3. 豫湘桂战役爆发后陷入全面危机

全面抗战路线和持久战的战略总方针

  1. 全面的全民族抗战路线（人民战争路线） - 1937年8月在陕北洛川召开政治局扩大会议，制定了抗日救国十大纲领，必须实行全国军事和全国人民的总动员
  2. 持久战的战略总方针
    1. 1938年发表《论持久战》，总结经验、驳斥亡国论和速胜论等错误观点，系统地阐明了持久战方针
    2. 持久战的根据 - 强国弱国的对比；日本是小国且发动的侵略战争，失道寡助
    3. “兵民是胜利之本”，必须进行人民战争
    4. 抗日战争的三个阶段 - 战略防御、战略相持、战略反攻。其中战略相持阶段是决定胜利最关键的阶段

敌后战场的开辟与游击战争的发展

  1. 敌后战场的开辟和发展
    1. 第一次重大胜利 - 平型关大捷
    2. 发动独立自主敌后游击战争 - 1937年太原失守。在华北以国民党为主体的正规战争结束、以共产党为主体的游击战争上升到了主要地位
    3. 建立抗日根据地，发动抗日武装
  2. 游击战争的战略作用和地位
    1. 战略防御阶段是辅助的
    2. 战略相持阶段是主要的
    3. 为人民军队进行战略反攻做了准备

坚持抗战、团结、进步的方针

  1. 统一战线中的独立自主原则
  2. 坚持抗战、团结、进步；反对妥协、分裂、倒退(1941年皖南事变)
  3. 巩固抗日民族统一战线的策略总方针 - 发展进步势力、争取中间势力、孤立顽固势力
  4. 对顽固派贯彻又联合又斗争的政策 - 坚持有理、有据、有节的原则

抗日民主根据地的建设

  1. “三三制的民主建设” - 加强政治建设是抗日根据地建设的首要的，根本的任务
  2. 抗日民主政权普遍采用民主集中制
  3. 民族区域自治是一个创造
  4. 减租减息政策与大生产运动
  5. 文化建设与干部教育
  6. 大后方的抗日民主活动和进步文化工作
    1. 1939年宪政活动；1944年林伯渠提出结束国民党一党统治
    2. 抗日文化工作的开展

中国共产党自身的建设

  1. 1938年党的六届六中全会
    1. “决定中国之命运”
    2. 毛泽东明确提出马克思主义中国化的命题
    3. 纠正了王明的右倾错误，巩固了毛泽东的领导地位，统一了全党的思想步调、
  2. 新民主主义理论的系统阐明
    1. 中国半殖半封呢个的性质和主要特征，明确近代中国社会的主要矛盾和中国革命发生及发展的原因。整个中国共产党领导中国革命包括民主主义革命和社会主义革命
    2. 阐明了中国共产党在新民主主义革命阶段的基本纲领
    3. 总结了党成立以来的历史经验 - 统一战线、武装斗争、党的建设
    4. 标志着毛泽东思想得到多方面的展开而趋于成熟
  3. 整风运动和《关于若干历史问题的决议》
    1. 整风运动的开展及其必要性 - 1941《改造我们的学习》；1942《整顿党的作风》《反对党八股》
    2. 整风运动的主要内容 - 反对主观主义以整顿学风(最主要任务)、反对宗派主义以整顿党风、反对党八股以整顿文风
    3. 1944-1945扩大的六届七中全会通过了《关于若干历史问题的决议》 - 对建党以后尤其是六届四中全会至到遵义会议的历史作总结，为七大的召开创造了充分条件。整风运动胜利结束
    4. 整风运动是一次深刻的马克思主义思想教育运动，也是一场思想解放运动。它坚持马克思主义同中国实际相结合的正确方法，使实事求是的思想路线在全党确立起来
  4. 中共七大和毛泽东思想的确立
    1. 19456年在延安召开
    2. 制定了党的政治路线，即放手发送群众、壮大人民力量、在我党的领导下，打败日本侵略者、解放全国人民、建立一个新民主主义的中国
    3. 首次在文件上明确提出 - 一切政策的好坏归根到底在于对中国人民生产力的发展
    4. 概括了三大作风 - 理论与实践相结合、和人民群众紧密联系在一起、自我批评
    5. 中共七大将毛泽东思想确定为党的指导思想
    6. 中共七大标志着党在政治上思想上组织上走向成熟

抗日战争的胜利

  1. 1945年7月26日，《波兹坦公告》敦促日本投降并归还台湾及澎湖列岛
  2. 8月15日无条件投降，9月2日在投降书上签字。抗日战争和世界反法西斯战争结束
  3. 抗日战争取得完全胜利的重要标志 - 归还台湾和澎湖列岛
  4. 世界反法西斯战争的东方主战场
  5. 中国人民抗日战争在世界反法西斯战争中的地位
  6. 抗日战争胜利的原因
    1. 以爱国主义为核心的民族精神是中国人民抗日战争胜利的决定因素
    2. 中国共产党的中流砥柱作用
    3. 全民族抗战是重要法宝
    4. 同国际反法西斯联盟的支持离不开
  7. 抗日战争胜利的意义
    1. 是近代中华民族反抗外敌入侵第一次取得完全胜利的民族解放战争，是中华民族近代以来陷入深重危机走向伟大复兴的历史转折点
    2. 彻底粉碎了日本军国主义殖民奴役中国的图谋
    3. 促进了中华民族的伟大团结，形成了伟大的抗战精神 - 天下兴亡、匹夫有责的爱国情怀；视死如归、宁死不屈的民族气节；不畏强暴、血战到底的英雄气概；百折不挠、坚忍不拔的必胜信念
    4. 对世界反法西斯战争胜利产生巨大影响，中国国际地位显著提高
    5. 开启历史新征程，为赢得新民主主义革命的胜利奠定了重要基础
