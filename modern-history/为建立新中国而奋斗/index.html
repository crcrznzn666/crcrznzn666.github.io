为建立新中国而奋斗

发表于 2025-08-19 更新于 2025-08-20 分类于 modern-history

1

历史

战后国内形式

  1. 国内国际整治形式
    1. 国民党统治集团根本目的是战后维持蒋介石的独裁统治，继续走半殖民地半封建的老路。以武力消灭中国共产党及其领导的人民军队和解放区政权是蒋介石集团的既定方针
    2. 美国采取扶蒋反共策略
  2. 中国共产党争取和平民主的方针
    1. 争取和平发展的新阶段
      1. 通过和平的途径对中国进行政治社会的改革，进行经济建设。这在当时既有必要也有可能
      2. 中国人民战争后和和平建国的渴望
      3. 人民力量强大，其他条件加持下和平发展新阶段的可能是存在的
    2. 1945年《对目前时局的宣言》中提出“民主、和平、团结”的口号
  3. 重庆谈判 - 签订《双十协定》。确定和平建国的基本方针，同意“长期合作，坚决避免内战”
  4. 1946年政治协商会议 - 下达停战令。会议达成了包括政府组织案、国民大会案、和平建国纲领、军事问题案、宪法草案案等五项协议
  5. 维护和破坏政治协议的较量
    1. 中国共产党准备严格履行政治协议
    2. 国民党统治集团从来没有准备去履行政治协议
    3. 中国共产党争取和平民主努力的意义

国民党发动全面内战和解放区军民的坚决反击

  1. 1946年6月26日，国民党军以进攻中原解放区为起点，挑起了全面内战
  2. 以革命战争反对反革命战争 - 中国共产党清楚认识到我们必须能打败蒋介石，而且能够打败蒋介石
  3. 以自卫战争粉碎国民党的军事进攻
    1. 政治军事双管齐下
      1. 政治上和人民群众亲密合作，争取一切可以争取的人，在党的领导下建立最广泛的人民民主统一战线
      2. 军事集中优势兵力，各个歼灭敌人的作战原则
    2. 粉碎国民党的全面进攻和重点进攻。
      1. 1946年6月至1947年6月的一年时间里，人民军队处于战略防御阶段，战争主要在解放区进行。
      2. 其中，1946年6月至9月，国民党军对解放区发动全面进攻
      3. 到1946年10月，被迫改为对陕北、山东两解放区的重点进攻
      4. 从1947年3月至6月，解放军努力打退国民党军的重点进攻，并在多地开启了局部反攻
      5. 从根本上粉碎了国民党统治集团的速战速决计划
  4. 解放战争的胜利发展
    1. 1947年中共中央立即决定转入全国性反攻，将战争引向国民党统治区域
    2. 三路大军的挺近揭开了人民战争战略进攻的序幕、
    3. 各个战场上上的攻势作战，构成了人民解放军全国规模的战略进攻的总形式
    4. 1947年10月10日，提出“打倒蒋介石，解放全中国”的口号

解放区的土地改革和农民的广泛发动

  1. 1946年发布《关于土地问题的指示》（五四指示） - 将抗日战争时期的减租减息变为耕者有其田。由削弱封建剥削向废除封建剥削过渡
  2. 1947年《中国土地法大纲》 - 规定废除封建性即半封建性剥削的土地制度，实行耕者有其田的土地制度。这是一个彻底的反封建的土地革命纲领

第二条战线的形成与发展

  1. 国民党统治地区的政治经济危机
  2. 学生运动的高涨
  3. 人民民主运动的发展
  4. 各民主党派的历史发展
    1. 少数成立于大革命时期与土地战争革命时期，多数成立于抗日战争和解放战争时期
    2. 主要是民族资产阶级、城市小资产阶级以及同这些阶级相联系的知识分子和其他爱国青年，是阶级联盟性质的政党
    3. 民主党派都是主张爱国民主的党派，中国共产党和民主党派的合作
    4. 第三条道路(中间路线)实际是资产阶级共和国方案，主张的实质是旧民主主义的道路
  5. 多党合作和政治协商格局的形成
    1. 1948年五一口号 - 揭开了中国共产党各党派、各团体、各族各界人士协商建国的序幕，奠定了中国共产党领导的多党合作2和政治协商的基础
    2. 1949年《对时局的意见》 - 表明中国各民主党派和无党派人士公开自愿接受中国共产党的领导，决心走人民革命的道路，拥护建立人民民主的新中国
    3. 1949年毛泽东发表讲话“民主党派积极参政，共同建设新中国” - 民主党派参加新政协并将在新中国参政，标志着明珠党派地位的根本变化。中国共产党领导的多党合作政治格局正是在此基础上形成的

南京国民党政权的覆灭

  1. 辽沈、淮海、平津三大战役 - 国民党主要军事力量基本被摧毁
  2. 1949年南京被占领，宣告国民党反动统治的覆灭

为新中国绘制蓝图

  1. 1948年中共中央政治局会议，毛泽东论述了国体(人民民主专政)和政体(人民代表大会制度)
  2. 中共七届二中全会 - 指出中国由农业国转向工业国，由新民主主义社会转变为社会主义社会的发展方向。毛泽东提出两个务必的思想，即务必使同志们继续地保持谦虚、谨慎、不骄不躁的作风、务必使同志们继续保持艰苦奋斗的作风
  3. 1949年由西柏坡向北平进发，比作进京赶考
  4. 中央领导人与中央机关、人民解放军总部进驻北平香山，标志着中国革命中心由农村转向城市
  5. 1949年《论人民民主专政》 - 向全国人民公开阐述了党在建立新中国问题上的主张，为新中国建立奠定了理论基础与政策基础
  6. 2和5共同构成了《中国政治协商会议共同纲领的基础》
  7. 1949年第一届中国人民政治协商会议的召开
    1. 通过三个重要文件
    2. 做出四个重要决议
    3. 选出政治协商会议全国委员会和中央人民政府委员会，宣告中华人民共和国的成立
    4. 标志的中国共产党领导的多党合作和政治协商制度的确立
  8. 《共同纲领》是当时的中国人民的大宪章，在一个时期内起到新中国临时宪法的作用

中国革命的总结

  1. 胜利原因
    1. 深刻的社会根源和雄厚的群众基础
    2. 根本上说是由于有了中国共产党的领导
    3. 同无产阶级和人民群众的支持是离不开的
  2. 伟大意义
    1. 第一，中国革命的胜利，结束了100多年来中华民族遭受资本-帝国主义侵略和中国各族人民遭受资本-帝国主义同封建统治阶级联合压迫与剥削的历史，结束了国家战乱频仍、四分五裂的局面，实现了中国人民梦寐以求的民族独立和人民解放。
    2. 第二，中国革命的胜利，从根本上改变了中国社会的发展方向，为实现由新民主主义到社会主义的转变和建立社会主义制度、进行社会主义现代化建设，扫清了主要障碍，创造了政治前提；为实现国家富强和人民幸福，实现中华民族伟大复兴，开辟了广阔道路。
    3. 第三，中国革命的胜利，是继俄国十月社会主义革命和世界反法西斯战争胜利后世界历史中最重大的事件。它在一个人口占全人类近1/4的大国里，冲破帝国主义的东方战线，极大改变了世界的政治格局，壮大了世界和平、民主和社会主义的力量，鼓舞了世界被压迫民族和被压迫人民争取解放的斗争，受到世界人民的欢迎和支持。
    4. 第四，中国人民革命的胜利，是在马克思列宁主义的指导下取得的。中国共产党创造性地运用马克思列宁主义的基本原理，把它同中国革命具体实际结合起来，形成了伟大的毛泽东思想，找到了夺取中国革命胜利的正确道路。这对于马克思列宁主义的发展是一个重大的贡献。
  3. 基本经验
    1. 必须坚持把马克思列宁主义的基本原理同中国的具体实际相结合
    2. 统一战线、武装斗争、党的建设是中国革命战胜敌人的三个法宝，三个主要的法宝
    3. 革命的根本问题是国家政权问题，集中到一点就是工人阶级(经过中国共产党)领导的以工农联盟为基础的人民民主专政
