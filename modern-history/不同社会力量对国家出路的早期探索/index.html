不同社会力量对国家出路的早期探索

发表于 2025-08-16 分类于 modern-history

1

历史

太平天国运动 1851-1864

  1. 爆发的原因
    1. 清政府加重了赋税的征收科派
    2. 西方资本主义入侵使中国的农业和家庭手工业相结合的自然经济逐渐解体
  2. 历程
    1. 创立拜上帝教
    2. 金田起义后迅速发展
    3. 占领南京、定为首都、改名天京，正式宣告太平天国农民政权的成立
    4. 北伐西征、达到军事上的全盛期
  3. 纲领性文件
    1. 《天朝田亩制度》
      1. 最能体现太平天国理想和此次农民起义特色的纲领性文件
      2. 定义是以解决土地问题为中心的涉及政治、经济、军事、文化内容的革命纲领
      3. 确立平均分配土地的方案，建立四有的理想社会
      4. 进步性 - 否定封建土地所有制，体现广大农民平分土地的强烈愿望，均贫富的思想具有进步性
      5. 空想性 - 因此最终未能实施
    2. 《资政新篇》
      1. 后期颁布的社会实施方案
      2. 政治方面 - 禁朋党之弊，加强中央集权学习西方法律制度
      3. 经济方面 - 从学习生产力拓展到学习生产关系，开始提倡资本主义的雇佣劳动制
      4. 文化方面 - 主张设立新闻管，兴办学校医院等社会福利事业
      5. 外交方面 - 同外国平等交往但严禁鸦片输入
      6. 评级是一个具有资本主义色彩的方案
  4. 历史意义
    1. 沉重打击了封建统治阶级
    2. 是中国旧式农民战争的最高峰，《资政新篇》由于学习了发展资本主义的方案，因此也具有了不同于以往农民战争的新的历史起点
    3. 冲击孔子正统权威、一定程度上削弱了封建统治的精神支柱
    4. 有力打击了外国侵略势力
    5. 在亚洲民族解放运动中也有浓墨重彩的一笔
  5. 失败原因
    1. 客观原因 - 中外反动势力的联合绞杀
    2. 主观原因(根本原因) - 农民阶级自身的局限性
    3. 农民不是新的生产力和生产关系的代表、无法克服小生产者固有的阶级局限性
    4. 教训是单纯的农民战争不可能完成争取民族独立和人民解放的历史任务

洋务运动

  1. 时间为二次鸦片战争结束后，清政府在镇压太平天国运动过程中兴起的
  2. 目的 - 维护清政府的统治、镇压农民起义、增强海防、增强本集团经济政治军事实力
  3. 指导思想 - 中体西用，即以中国封建伦理纲常为主体、西方近代工业和技术为辅助
  4. 目标 - 前期自强；后期求富
  5. 主要活动
    1. 兴办近代企业
      1. 军用工业 - 形式都是官办的
      2. 民用企业 - 多数采用官督商办的形式，但基本都是资本主义的近代企业
    2. 建立新式海陆军
    3. 创办新式学堂、派遣留学生
  6. 失败
    1. 甲午中日海战中北洋海军全军覆没，标志着洋务运动的失败
    2. 原因
      1. 封建性 - 网图维护封建统治
      2. 对列强具有依赖性
      3. 管理具有腐朽性
  7. 历史意义
    1. 客观上对中国早期工业和民族资本主义发展起到了促进作用
    2. 近代教育得以开始，开阔了人们的眼界
    3. 社会风气改变，工商业者地位提升，西方的器物技术等不再被排斥，有利于资本主义经济的发展
  8. 启示 - 维持封建的上层建筑、经济基础前提下的自强求富，这条道路是走不通的

戊戌变法

  1. 历史条件
    1. 1890年后中国民族资本主义有了初步发展
    2. 甲午战争惨败造成了新的民族危机、激发了新的民族觉醒
    3. 代表民族资本主义发展要求的知识分子站在救亡图存和变法维新的前列。要求不但学习西方科学技术还要学习制度和思想
    4. 形成共识，要救国只有维新，要维新只有学外国
  2. 维新派的活动
    1. 向皇帝上书
    2. 著书立说
    3. 介绍外国变法的经验教训
    4. 办学会、设学堂、办报纸
    5. 重点放在争取光绪及其周围官员的支持，希望通过他们实现自上而下的变法主张
  3. 维新派和守旧派的论战(实质上是资产阶级思想和封建主义思想在中国的第一次交锋)
    1. 要不要变法
    2. 要不要兴民权、设议院、实行君主立宪
    3. 要不要废八股、改科举、兴西学
  4. 百日维新的时间与主要内容，戊戌政变标志着戊戌变法的失败
  5. 意义
    1. 爱国救亡运动
    2. 资产阶级性质的政治改良运动
    3. 思想启蒙运动
  6. 失败原因 - 主要是由于维新派自身局限性和以慈禧太后为首的强大守旧势力的反对
    1. 民族资产阶级力量弱小
    2. 维新派自身的局限性
      1. 不敢否定封建主义，妄想通过和平改良实现君主立宪
      2. 对帝国主义抱有幻想
      3. 脱离了人居民群众甚至惧怕仇视
  7. 启示 - 在半殖民地半封建的中国，企图通过统治者走自上而下的改良道路是行不通的，必须用革命的手段
