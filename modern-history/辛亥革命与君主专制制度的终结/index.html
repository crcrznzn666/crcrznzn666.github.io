辛亥革命与君主专制制度的终结

发表于 2025-08-16 更新于 2025-08-17 分类于 modern-history

1

历史

辛亥革命的背景

  1. 历史条件
    1. 民族危机加深与社会矛盾激化
    2. 清末新政及其破产(根本目的在于延续反动统治)
    3. 资产阶级革命派的阶级基础和骨干力量 - 一批资产阶级和小资产阶级知识分子
  2. 资产阶级革命派的活动
    1. 孙中山与资产阶级民主革命的开始
      1. 孙中山走向了革命道路 - 根本上说，近代中国革命史外国侵略者和本国封建统治阶级逼迫出来的
      2. 1894年成立第一个革命团体兴中会。立誓驱逐鞑虏、恢复中华、创立合众政府。1904年发表《中国问题的真解决》一文，表明选择了武装推翻清政府统治的斗争方式，这是根本不同之处
    2. 资产阶级革命派的宣传组织工作
      1. 各种印象深刻的书名
      2. 成立革命团体 - 华兴会、光复会等
      3. 同盟会建立 - 以《民报》为机关报，确立了革命纲领(驱逐鞑虏，恢复中华)。这是近代中国第一个资产阶级革命的全国性政党.它的成立标志着中国革命进入新的阶段

三民主义的提出

  1. 民族主义
    1. 包含驱逐鞑虏恢复中华两项内容(民族革命)
    2. 以革命手段推翻清政府，改变民族歧视和民族压迫政策
    3. 建立中华民族独立的国家
    4. 没有正面鲜明鲜明提出反对帝国主义，甚至对帝国主义抱有幻想；放松了汉族封建势力
  2. 民权主义
    1. 创立民国，推翻君主专制制度，建立资产阶级民主共和国(政治革命)
    2. 忽略了广发劳动群众在国家中的地位，难以使人民的民主权益得到真正的保障
  3. 民生主义
    1. 指的是平均地权
    2. 但是没有将土地所有权分给农民，没有正面触及封建土地所有制，不能满足广大农民的土地要求，在革命中难以成为发动广大工农群众的理论武器
  4. 初步描绘了中国不曾有过的资产阶级共和国方案，是一个比较完整而明确的资产阶级民主革命纲领

关于革命与改良的辩论

  1. 论战主要内容 - 围绕中国究竟是采用革命手段还是改良方法。分别在《民报》《新民丛报开启论战》
  2. 要不要以革命手段推翻清王朝
  3. 要不要推翻帝制实现共和
  4. 要不要进行社会革命
  5. 划清了革命与改良的界限、传播了民主革命思想、促进了革命形势的发展
  6. 暴露了革命派在思想理论方面的弱点，如对帝国主义的态度、国民的定义等

辛亥革命的历史进程

  1. 辛亥革命的爆发与清王朝覆灭
    1. 同盟会的先后起义，最著名的黄花岗起义(广州起义)
    2. 保路运动的兴起
    3. 武昌起义拉响了中国完全意义上近代历史民主民族革命的的序幕
    4. 1912年2月12如，清帝退位，封建君主专制制度终于覆灭
    5. 革命内部和外部同时潜伏者深刻危机
  2. 中华民国的成立
    1. 1912年为民国元年，成立中华民国临时政府
    2. 南京临时政府是资产阶级共和国性质的革命政权
    3. 南京政府的局限性表现在承认不平等条约来换取成人；没有提出任何可以满足农民土地要求的政策和措施，反而以保护私有财产为借口维护封建土地制度
  3. 《中华民国临时约法》
    1. 中国历史上第一部具有资产阶级共和国性质的法典
    2. 主权属于全体国民，以参议院、临时大总统、国务员、法院行使统治权
    3. 大总统代表临时政府总览政务
    4. 中华民国国民一律平等
    5. 以根本大法的形式废除了两千多年来的封建君主专制制度，确立了资产阶级共和国的政治制度，带有鲜明的革命性、民主性和进步性
  4. 历史意义
    1. 比较完全意义上的资产阶级民主革命
    2. 推翻封建示例的政治代表，帝国主义在中国的代言人清王朝的统治，沉重打击了中外反动势力
    3. 结束了两千年的封建君主专制制度，建立了历史第一个资产阶级共和政府，民族共和观念开始深入人心
    4. 推动了中国人民的思想解放
    5. 推动社会变革、解放社会风气
    6. 一定程度上大几帝国主义的侵略气焰、推动亚洲民族解放运动的高潮

封建军阀专职统治的形成

  1. 同年袁世凯窃取革命果实
  2. 北洋军阀的专制统治
    1. 在北京建立以代表大地主和买办资产阶级力易得北洋军阀反动统治
    2. 实施军阀官僚的专制统治；承认21条；复辟封建帝制
    3. 接力维护帝国主义、地主阶级和买办资产阶级的利益
    4. 尊孔复古潮流猖獗一时
  3. 袁世凯死后中国陷入军阀割据，但是形式上仍是统一的

旧民主主义革命的失败

  1. 二次革命的失败
  2. 护国运动逼迫袁世凯取消帝制
  3. 护法运送失败，依靠南方军阀反对被与昂军阀是行不通的
  4. 辛亥革命失败的原因及教训
    1. 根本上说，在半殖半封的中国社会，资本主义的建国方案是行不通的
    2. 主观上，领导者与参与者存在许多弱点与错误
      1. 未能提出彻底的反帝反封建的革命纲领
      2. 不能充分发动和依靠人民群众
      3. 不能建立坚强的革命政党，作为团结一切革命力量的强有力的核心
    3. 辛亥革命的弱点，根源于民族资产阶级的软弱性和妥协性，无法完成反帝反封建的任务。资产阶级共和国的方案没有能够救中国
