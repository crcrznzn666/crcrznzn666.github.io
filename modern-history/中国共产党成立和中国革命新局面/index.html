中国共产党成立和中国革命新局面

发表于 2025-08-17 分类于 modern-history

1

历史

新文化运动和思想解放的潮流

  1. 新文化运动
    1. 背景 - 辛亥革命失败与北洋军阀的统治，决心发动新的启蒙运动接发人们封建的思想
    2. 1915年陈独秀创立《新青年》杂志，标志着新文化运动的开始。《新青年》和北京大学成为新文化运动的主要阵地
    3. 基本内容 - 提倡民主科学，提倡个性解放，提倡新道德等
    4. 基本口号 - 提倡民主和科学
    5. 评价
      1. 性质 - 资产阶级民主主义的新文化反对封建主义旧文化的斗争
      2. 新文化运动的重大历史意义
      3. 局限性
        1. 批判孔学是为了给资本主义发展扫清障碍，但是资本主义并不能为人们提供一种新的思想武器去认识中国
        2. 新文化运动的倡导者把改造国民性置于优先地位。但是离开改造封建环境的革命实践，要改造这种国民性是不可能的
        3. 部分领导者没有马克思的批判精神，看待问题过于片面

十月革命与马克思主义在中国的初步传播

  1. 经济落后的国家也可以用社会主义思想指引自己走向解放之路
  2. 诞生地俄国号召反对帝国主义，并以新的平等态度对待中国，有力地推动了社会主义在中国的传播
  3. 十月革命那中俄国工人、农民和士兵群众的广泛发动并由此赢得胜利的事实，基于中国新进分子以新的革命方式的启示
  4. 中国举起马克思主义旗帜第一人 - 李大钊
    1. 《法俄革命之比较观》 - 第一次向中国人民正确阐述了十月革命的性质、
    2. 《庶民的胜利》，《Bolshevism的胜利》‘
    3. 《我的马克思主义观》明确把马克思主义称为“世界改造原动的学说”

五四运动：新民主主义革命的开端

  1. 五四运动爆发的时代条件和历史条件
  2. 巴黎和会上中国外交的失败是五四运动的直接导火索
  3. 两个阶段
    1. 5.4-6.5，以学生为主，中心在北京
    2. 6.5-，以工人为主，中心在上海。工人阶级以独立的姿势登上政治舞台
  4. 罢免亲日官僚、拒绝出席巴黎合约的签字，五四运动直接的斗争目标得以实现
  5. 历史特点 - 爱国革命运动、社会革命运动、思想启蒙运动和新文化运动
  6. 意义
    1. 旧民主主义革命走向新民主主义革命的转折点，在近代以来中华民族追求民族独立和发展进步的历史进程中具有里程碑意义。五四运动成为了中国新民主主义革命的开端
    2. 以全民族的力量高举爱国主义的伟大旗帜，孕育了爱国、进步、民主、科学为主要内容的伟大五四精神，其核心是爱国主义
    3. 以全民族的行动激发了只求真理、追求进步的伟大觉醒，实现了中国人民和中华民族自鸦片战争以来第一次全面觉醒
    4. 以全民族的搏击培育了永久奋斗的伟大传统

中国早期马克思主义运动

  1. 早期马克思主义者的队伍
    1. 新文化运动的精神领袖 - 李大钊陈独秀等
    2. 五四爱国运动的左翼骨干 - 毛泽东等
    3. 一部分原同盟会成员，辛亥革命时期活动家 - 董必武等
  2. 早期马克思主义思想运动的特点
    1. 重视对马克思主义基本理论的学习
    2. 注意从中国实际出发、学习和应用马克思主义
    3. 开始提出知识分子应与劳动群众相结合的思想
  3. 新文化运动的发展
  4. 马克思主义与工人运动的结合
    1. 中国共产党早期组织 - 由陈独秀和李大钊最先酝酿，在中国工人阶级最密集的中心城市上海建立
    2. 中国共产党早期组织的活动
      1. 研究和宣传马克思主义，研究中国实际问题
      2. 到工人中进行宣传和组织工作
      3. 进行关于建党问题的讨论合适组织工作
  5. 中国共产党第一次全国代表大会的召开和中国共产党的成立
    1. 中国共产党的诞生成为历史发展的必然（三个必然点）
    2. 中国共产党第一次全国代表大会
      1. 确立党的名称为中国共产党
      2. 通过党的第一个纲领，明确革命军队必须与无产阶级一起推翻资产阶级的政权；承认无产阶级专政知道阶级斗争结束；消灭资本家私有制；联合第三国际
      3. 旗帜鲜明地把社会主义和共产主义规定为自己的奋斗目标，坚持用革命的手段实现这个目标，决定集中力量组织工人
      4. 设立中央局作为中央的临时领导机构，选举陈独秀作为中央局书记
    3. 中国共产党成立的历史特点
      1. 成立于十月革命胜利，第二国际破产。接受的是完整的科学世界观和社会革命论的马克思主义
      2. 在半殖民地半封建的工人运动基础上产生的，具有坚强的革命性
      3. 一开始就一个以马克思鬣羚注意为基础的党，是一个区别于第二国际旧式社会改良党的新兴工人阶级革命政党
    4. 中国共产党成立的历史意义
      1. 开天辟地的大事变，具有伟大而深远的意义
      2. 从根本上改变了没有先进的坚强的政党作为凝聚力量的领导核心的局面
      3. 把实现共产主义作为党的最高理想和最终目标，肩负实现中华民族伟大复兴的历史使命
      4. 中国人民由此踏上了光明道路，开启了历史征程
      5. 三个深刻改变
      6. 形成了坚持真理、践行初心、担当使命、不怕牺牲、英勇斗争、对党忠诚、不负人民的伟大建党精神，这是中国共产党的精神之源

民主革命纲领的制定和工农运动的发动

  1. 民主革命纲领
    1. 中共二大首次提出了反帝反封建的民主革命纲领。分清敌友，这是革命的首要问题
    2. 党的最高纲领是实现社会主义、共产主义。但是目前最低纲领是达到军阀；推翻国际帝国主义的压迫，统一中国为真正的民主共和国
    3. 发动工农群众开阵革命斗争
  2. 工农运动的发动
    1. 1922年香港海员罢工为起点，1923年京汉铁路工人罢工为终点，掀起了工人运动第一个高潮
    2. 安源路矿工人大罢工是中国共产党第一次独立领导并取得完全胜利的工人斗争
    3. 1921年9月，浙江萧山县衙前村成立中国第一个农村协会
    4. 青年运动和妇女运动的开展
  3. 国共合作和大革命的进行
    1. 中国共产党从实践中认识到了建立革命统一战线的重要性 - 1923年中共三大，明确共产党员以个人身份加入国民党以实现国共合作
    2. 国民党一大标志着第一次国共合作的正式形成。事实上确立了联俄、联共、扶助农工三大革命政策
    3. 1925年5月，以五卅运动为起点，掀起了全国范围内大革命高潮
    4. 同年6月的省港大罢工是中国工人历史上持续时间最久的一次政治大罢工
    5. 1925年7月1日国民政府在广州建立
    6. 北伐战争的胜利进展
    7. 大革命中的中国共产党有着不可替代的作用
      1. 大革命的口号是中国共产党提出的 - 反对帝国主义、反对军阀
      2. 是在国共合作的基础的统一战线的组织形势下进行的
      3. 大革命是近代史上空前广泛而深刻的群众运动。而中国共产党正是人民群众的主要发动者和组织者
      4. 大革命的主要斗争形式是革命战争
    8. 大革命的失败 - 1927年国民革命军占领南京后受到炮轰-1927年蒋介石发动412反革命政变-同年7月15日汪精卫武汉召开分共会议
      1. 失败原因
        1. 客观原因 - 反革命力量强大；资产阶级严重动摇；汪蒋集团的叛变
        2. 主观原因 - 中国共产党处在幼年时期缺乏经验，缺乏对革命的认识，缺乏马克思主义基本原理同中国革命具体实际结合；党内陈独秀为首的右倾错误
        3. 国际原因 - 大革命后期共产国际的错误指导
      2. 经验教训
        1. xxx和中国具体实际相结合
        2. 不仅要统一战线，还要保持独立性，做到又团结又斗争的方针
        3. 必须武装斗争
        4. 必须解决农民的土地问题
        5. 加强党的自身建设，加强党的民主集中制。既要注重党员数量，更要巩固党员质量
      3. 历史意义
        1. 领导了轰轰烈烈的革命工作，领导了全国反帝反封建的伟大狗证
        2. 开始了探索马克思主义中国化的途径，初步提出了新民主主义革命的基本思想
        3. 汲取教训，开始懂得土地革命和掌握革命武装的重要性
        4. 为土地革命的战争阶段做了充足准备
