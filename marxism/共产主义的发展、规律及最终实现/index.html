共产主义的发展、规律及最终实现

定论

  1. 发展阶段 - 16-17世纪早期空想社会主义；18世纪空想平均社会主义；19世纪初期批判的空想社会主义(科学社会主义直接的思想来源)
  2. 空想社会主义的历史贡献与局限性
  3. 科学社会主义的创立与意义

十月革命与第一个社会主义国家的建立

  1. 一国胜利论 - 社会主义可能首先在少数甚至单独一个资本主义国家内取得胜利
  2. 社会主义在苏联的实践
  3. 列宁对建设社会主义的新构想
    1. 建设社会主义是一个长期探索、不断实践的过程
    2. 大力发展生产力、提高劳动生产率放在首位
    3. 多种经济成分并存，利用商品、货币和市场发展经济
    4. 利用资本主义建设社会主义
  4. 苏联模式的形成与特征

社会主义的发展与观点

  1. 社会主义发展到多个国家
  2. 社会主义在中国焕发出蓬勃生机
  3. 科学社会主义基本原则的主要内容
    1. 资本主义必然灭亡，共产主义必然胜利是科学社会主义的核心命题
    2. 共产主义的实现是一个长期的过程，需要不断的实践和探索
    3. 无产阶级是最先进最革命的阶级
    4. 无产阶级反抗资产阶级的斗争有三种形式:经济斗争、政治斗争和思想斗争。其中政治斗争有最关键的意义。无产阶级革命根本问题是国家政权问题
  4. 正确把握科学社会主义基本原则
    1. 正确对待科学社会主义基本原则的态度
    2. 中国特色社会主义的特点
    3. 社会主义建设的长期性
    4. 社会主义发展道路的多样性
    5. 社会主义在实践中开拓前进

共产主义的崇高理想及其最后实现

  1. 预见未来社会的方法论原则
  2. 共产主义社会的基本特征
    1. 物质财富极大丰富，消费材料按需分配
    2. 社会关系高度和谐，人们精神境界极大提高
    3. 实现人自由而全面的发展，人类必须将从必然王国向自由王国飞跃实现人自由而全面的发展，是马克思主义追求的根本价值目标，也是共产主义的根本特征
  3. 实现共产主义是历史发展的必然
  4. 实现共产主义是长期的历史进程
  5. 坚持理想与共同理想的辩证统一

混淆

    1. 1864年第一国际促进了马克思主义及工人运动的结合
    2. 1871年巴黎公社运动是无产阶级夺取政权的第一次伟大尝试
    3. 十月革命实现了社会主义从理想到现实的伟大飞跃，开拓了人类新纪元
  2. 社会主义在苏联的实践
    1. 1917年到1918年 - 进一步巩固苏维埃政权时期
    2. 1918年到1921年 - 战时共产主义，实行余粮征集制和取消商品货币关系为主的战时经济政策
    3. 1921年之后 - 新经济政策时期，用粮食税制取代余粮征集、恢复货币关系、允许私人小工业企业发展，采取国家资本主义形式发展生产
