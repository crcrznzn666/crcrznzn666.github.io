资本主义的发展及其趋势

发表于 2025-07-18 分类于 marxism

资本主义的本质及发展规律

定论

垄断

  1. 19世纪70年代开始，自由资本主义逐步向垄断资本主义发展。垄断资本主义包括私人垄断资本主义和国家垄断资本主义。其中私人垄断资本主义是在生产集中和资本集中的基础上形成的
  2. 1. 生产集中 - 生产资料、劳动力和商品日益集中于大企业
2. 资本集中 - 大资本吞并小资本，或小资本合并为大资本
3. 上述两项<font color = purple>是资本家追逐剩余价值的结果</font>
  3. 垄断的形成、本质及垄断组织
  4. 垄断不能消除竞争
    1. 垄断没有消除产生竞争的经济条件
    2. 垄断必须通过竞争维持
    3. 任何垄断组织都不能把所有社会生产全部包下来
  5. 同自由竞争相比，垄断在目的、手段和范围上具有新特点
  6. 金融资本与金融寡头
    1. 金融资本是工业垄断资本和银行为垄断资本融合形成的一种垄断资本。形成的主要途径包括金融联系、资本参与和人事参与
    2. 金融寡头是操纵国民经济命脉，并在实际上控制国家政权的少数垄断资本家或垄断资本家集团。在经济领域上通过“参与制”实现统治，在政治领域通过同政府的“个人联合”实现统治
  7. 垄断利润和垄断价格
    1. 垄断资本的实质是获取垄断利润
    2. 垄断利润归根到底来自于剩余价值
    3. 垄断利润主要是通过垄断高价(卖出)和垄断低价(购入)实现的
    4. 垄断价格的产生并没有违背价值规律,实质上是对商品价值和剩余价值做了有利于垄断资本的再分配
  8. 国家垄断资本主义
    1. 有着缓和社会矛盾，调节利益需求的作用
    2. 五种形式 - 直接掌控、与私人共有、多种形式参与私人垄断再生产、宏观调节、微观规制(反托拉斯法、公共事业规制、社会经济规制)
    3. 双重作用
      1. 对资本主义经济发展产生了积极作用。一定程度上有利于社会发展，对劳动人民生活有所改善
      2. 加大了对劳动人民的剥削掠夺，实质上是私人垄断资本利用国家机器来为其服务的手段，是私人垄断资本与国家政权结合的新垄断资本主义形式
  9. 金融自由化和金融创新推动资本主义经济的金融化程度不断提高
  10. 垄断资本向世界范围的扩展及后果
  11. 垄断资本国际化条件下的垄断组织 - 早期形式-国际卡特尔；当代形式 - 国家垄断资本主义的国际联盟

经济全球化

  1. 含义
  2. 表现为 - 生产全球化、贸易全球化和金融全球化
  3. 成因
    1. 经济全球化是生产力发展和社会化大生产的必然要求
    2. 科学技术进步与生产力的发展
    3. 扩过公司的发展
    4. 各国经济体制的变革和国际经济组织的发展
  4. 影响
    1. 是生产社会化程度不断提高的过程
    2. 发达国家和发展中国家在全球化进程中的地位和收益不平等

资本主义的变化及实质

  1. 资本主义发展初期,私人资本所有制占主导地位，19世纪末，私人股份资本所有制成为占主导的所有制形式,第二次世界大战后，国家资本所有制形成并发挥重要作用，法人资本所有制崛起并成为居主导的资本所有制形式
  2. 劳资关系和分配关系的变化 - 包括职工参与决策、终身雇佣、职工持股
  3. 社会阶级和阶层的变化
    1. 资本家地位和作用发生很大变化
    2. 高级职业经理成为大公司经营活动的实际控制者
    3. 知识型和服务型劳动者数量增加，劳动形式发生新变化
  4. 经济调节机制的变化与经济危机的新特点
    1. 二战结束后国家对经济全面干预，但是70年代后受新自由主义思潮影响，普遍走上弱化政府干预的道路
    2. 去工业化和产业空心化日趋严重、虚拟经济和实体经济严重脱节等特点
  5. 政治制度的变化
    1. 政治制度有多元化趋势，公民权益有所扩大
    2. 普遍加强了法治建设，以便协调社会各阶级阶层的利益
    3. 改良主义政党在政治舞台上影响日益扩大
  6. 变化的原因
    1. 科学技术革命和生产力发展，是根本推动力量>
    2. 工人阶级争取自身权利和利益的斗争是重要力量
    3. 社会主义制度初步展示优越性是重要影响
    4. 主张改良的政党对资本主义制度的改革也起了重要作用
  7. 变化实际上没有改变资本主义的基本矛盾
  8. 当代资本主义变化新特征
    1. 科技创新加速资本主义生产方式变化
    2. 国际金融垄断资本主义影响日益显现(最突出主要的特征)
    3. 社会阶级层级结构呈现复杂化多样化
    4. 发达资本主义凭借经济科技文化传播等优势，在世界推行霸权主义和强权政治
  9. 资本主义的历史地位与发展趋势
    (国家垄断资本主义是资本社会化的更高形式，将成为社会主义的前奏)
